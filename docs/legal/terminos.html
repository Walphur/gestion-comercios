Términos de uso

Gestión Comercios · Waltech · Última actualización: junio 2026

Al instalar y usar Gestión Comercios aceptás estos términos. Si no estás de acuerdo, no uses el software.

1. Licencia

Se otorga una licencia de uso no exclusiva y no transferible, según el plan adquirido (Básico: 1 PC; Pro: cantidad de PCs indicada en la compra). No se vende el código fuente.

2. Pago

Salvo que se indique lo contrario en la publicación de venta, el precio es pago único por licencia, no suscripción mensual obligatoria.

3. Activación

Cada licencia se activa con una clave y se vincula al equipo donde se activó por primera vez. Cambios de PC requieren contacto con soporte.

4. Requisitos técnicos

Windows 10 u 11 (64 bits). Internet necesario para la primera activación y actualizaciones. No compatible con Windows 7.

7. Soporte

8. Actualizaciones

Waltech puede publicar actualizaciones. Las correcciones de seguridad y mejoras menores se distribuyen sin cargo adicional salvo que se anuncie un producto nuevo de pago.

9. Limitación de responsabilidad

El software se ofrece “tal cual”. Waltech no se hace responsable por pérdida de datos por fallas de hardware, uso incorrecto o falta de copias de seguridad. Se recomienda usar las funciones de respaldo de la app.
